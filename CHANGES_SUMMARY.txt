5. HOW TO USE NETWORK RECEIVER
   Step 1: Start MediView Pro
   Step 2: Go to Settings > Network tab
   Step 3: Copy the IP:Port shown
   Step 4: Configure your USG machine with this IP:Port
   Step 5: Send DICOM files from USG - they'll appear in Network tab
   Step 6: Files are auto-saved in [AppData]/network-dicom/

6. NETWORK RECEIVER DETAILS
   - Port: 3458 (configurable in main.js DICOM_LISTEN_PORT)
   - Listen Address: 0.0.0.0 (all interfaces)
   - Storage: [UserData]/network-dicom/
   - File Naming: received_[timestamp].dcm
   - Auto-detection: Detects DICOM by magic bytes (0x28 or DICM signature)
